# ウデマエ - アーカイブ
ウデマエの上がり下がりを可視化する取り組みです。

個人用の記録アプリであり、だれでも使えるようなサービスにする予定は今のところないです。(人様のデータをあずかりたくないので・・)

勝手にForkして勝手にホストしてもらっても別に良いです。

## How to install
```sh
# git cloneして
npm i -g browserify
npm i -g node-sass
npm i
npm run build
```

ここまですればビルド完了です。

適当にサーバー立てて`index.html`へアクセスすれば見れます。
